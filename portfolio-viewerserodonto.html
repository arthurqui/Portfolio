Sobre o Viewer Serodonto

O Viewer Serodonto é uma aplicação web e PWA de alto desempenho voltada a dentistas referenciadores. Ele atua como um portal centralizado onde os profissionais podem visualizar de forma intuitiva exames odontológicos, laudos e imagens médicas encaminhados por clínicas de radiologia. Além disso, o sistema atua estrategicamente como um canal de aquisição (Funil de Vendas e Paywall) para o ERP parceiro de gestão odontológica.

O núcleo de visualização foi construído inteiramente no cliente utilizando VTK.js e ITK, permitindo renderização 3D nativa no navegador para modelos volumétricos (CEFX, DICOMs) e modelos STL, eliminando a dependência de servidores de renderização pesados e fornecendo uma experiência fluida. A aplicação conta com sincronização em tempo real entre abas via Redux State Sync e cache local usando PouchDB, garantindo operação contínua mesmo com oscilações de rede no consultório.

No lado do servidor, desenvolvi uma API robusta em .NET 6 (C#) baseada nos princípios de Clean Architecture e DDD, que se comunica de forma bidirecional com o sistema legado das clínicas via Azure Service Bus e integrações. O backend fornece notificações em tempo real com SignalR, serviços de e-mail (SendGrid) e WhatsApp automatizados para melhorar a comunicação com os pacientes e dentistas.

Stack tecnológica:
Frontend (SerFront): React 18, TypeScript, Redux Toolkit, React Query, PouchDB, VTK.js, Material-UI (MUI), PWA.
Backend (SerBack): .NET 6 Web API, PostgreSQL (Npgsql + EF Core 7), Clean Architecture / DDD, SignalR, Azure Service Bus, JWT, FluentValidation, Swagger.
