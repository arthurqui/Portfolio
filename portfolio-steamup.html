Sobre o SteamUp

O SteamUp é um marketplace especializado na comercialização de skins e produtos premium para a plataforma Steam. Desenvolvi este sistema focando em uma experiência de usuário fluida e segura para gamers que buscam itens exclusivos para jogos como CS:GO e Dota 2.

A plataforma utiliza a API oficial da Steam para autenticação via OpenID, garantindo que o inventário e os dados dos usuários sejam integrados de forma direta e confiável. Para a gestão financeira, implementamos o gateway do Mercado Pago, oferecendo diversas opções de pagamento com processamento rápido.

A arquitetura conta com um backend robusto em Python para processamento de dados e integração com a Steam, enquanto o frontend em React + Vite entrega uma interface reativa e de alta performance. O uso do Firebase permite atualizações em tempo real no marketplace e notificações instantâneas.

Stack tecnológica: React + Vite no frontend, Firebase para banco de dados e hosting, API em Python para lógica de negócio complexa, Gateway de pagamento Mercado Pago e integração com Steam API.
