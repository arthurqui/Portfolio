Tratamento e Análise de Dados Climatológicos

Durante minha bolsa de Iniciação Científica no **CPTEC/INPE** (Centro de Previsão de Tempo e Estudos Climáticos), fui responsável pelo desenvolvimento de um sistema robusto para o tratamento e análise de grandes volumes de dados climatológicos. O objetivo central era transformar dados brutos e massivos em informações visuais compreensíveis para auxiliar no monitoramento meteorológico nacional.

Utilizei a linguagem **NCL (NCAR Command Language)** e **Python** para criar scripts automatizados que realizavam o tratamento estatístico de dados provenientes de diversas fontes, como o dataset **CHIRPS** (Climate Hazards Group InfraRed Precipitation with Station data). Esses scripts processam arquivos de extensão **.nc (NetCDF)**, quantificando fontes de dados e gerando automaticamente gráficos complexos, mapas de calor e painéis de indicadores.

Os resultados obtidos foram validados através da comparação com dados públicos, comprovando a eficácia e precisão dos algoritmos desenvolvidos. O projeto teve um impacto direto na modernização de sistemas antigos do CPTEC-INPE, fornecendo ferramentas atualizadas para a previsão e o monitoramento de eventos meteorológicos extremos, como secas prolongadas e chuvas intensas.

Stack tecnológica: NCL (NCAR Command Language), Python, NetCDF (.nc), CDO (Climate Data Operators), Data Science e Estatística Climatológica.
