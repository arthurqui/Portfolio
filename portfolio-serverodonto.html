Sobre o ServerOdonto

O ServerOdonto é um robusto sistema de gestão (ERP) voltado para clínicas odontológicas, médicas e radiológicas. Com quase 20 anos de mercado, a plataforma abrange todo o ciclo de vida do paciente na clínica, desde o agendamento e atendimento clínico, até o faturamento, emissão de notas fiscais, cobrança via TEF/boletos e assinatura digital de documentos.

Durante minha atuação neste projeto maduro, desenvolvi uma forte capacidade de leitura, manutenção e evolução de código legado (programação "raiz", sem o auxílio de IA na época). Fui responsável por implementar novas funcionalidades, realizar correções críticas e gerenciar o deploy diretamente em servidores IIS, garantindo a estabilidade para centenas de clínicas.

Ajudei a manter e modernizar a arquitetura híbrida, integrando componentes WebForms consolidados com soluções modernas como ASP.NET SignalR para entrega de laudos em tempo real e Web API para integrações externas (NFS-e, WhatsApp, Cappta, DocuSign e aplicativos mobile).

Stack tecnológica: C# (.NET Framework 4.6), ASP.NET WebForms, Web API 5.2.4 (REST), SignalR 2.2.3, Telerik RadControls, Bootstrap, Firebird, Telerik OpenAccess ORM, e robustas integrações fiscais e financeiras.
